Where should the interest quiz appear?

The current post-signup path is: Sign Up → OrganizerCodeGate → Home. Where do we insert the quiz?

A

Right after OrganizerCodeGate → new route /interests

Sign Up → OrganizerCodeGate → Interest Quiz → Home
Catches all new users (both Google OAuth and email). First thing they do before seeing the app. Very high completion rate.

Pros

  • 100% exposure for new users
  • Personalizes dashboard from day 1

Cons

  • Adds friction before first dashboard view
B

As the final step inside OrganizerCodeGate (inline)

Sign Up → OrganizerCodeGate (Step 1: avatar + code | Step 2: Pick your interests) → Home
Keeps the post-signup flow as one unified screen/wizard.

Pros

  • Fewer routes/redirects
  • One cohesive "complete your profile" wizard

Cons

  • OrganizerCodeGate already handles avatar + code logic — adding interests bloats it
C

Deferred — prompted from Dashboard banner after first login

Sign Up → OrganizerCodeGate → Home → Banner: "Complete your profile → Pick interests"
Non-blocking. User sees the app first, then chooses to set preferences.

Pros

  • No signup friction
  • User can skip entirely

Cons

  • Lower completion rate
  • Dashboard won't be personalized on first visit
